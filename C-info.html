C Language

What is C language ?

C language is a general-purpose programming language developed by Dennis Ritchie in the early 1970s at Bell Labs. It was designed primarily for system programming to write operating systems such as Unix. C has since become one of the most widely used programming languages due to its efficiency, flexibility, and portability.

C has influenced many other programming languages, including C++, Java, and Python. It remains popular for system programming, embedded systems, and performance-critical applications. Learning C provides a solid foundation for understanding programming concepts and principles that are applicable across a wide range of domains.

Variables

Declaration:

Variables must be declared before they can be used. A declaration specifies the variable's name and its data type.

int count = 10 ; // Declaration and initialization of an integer variable 'count' with value 10

float pi = 3.14 ; // Declaration and initialization of a floating-point variable 'pi' with value 3.14

char grade = 'A' ; // Declaration and initialization of a character variable 'grade' with value 'A'

Here's an example demonstrating the usage of variables in C:

#include < stdio.h >

int main() {

int age = 25 ;

float weight = 65.5 ;

char grade = 'A' ;

printf ( "Age: %d\n", age ) ;

printf ( "Weight: %.2f\n", weight ) ;

printf ( "Grade: %c\n", grade ) ;

return 0 ;

}

  • double : Used to store floating-point numbers with double precision, typically 8 bytes.

  • long : Used to store integer values, typically 4 or 8 bytes, depending on the system.

  • long long : Used to store larger integer values, typically 8 bytes.

Control Structures

1. if-else Statement:

  • Allows you to execute a block of code conditionally based on whether a specified condition evaluates to true or false.

if (condition) {

// code block executed if condition is true

} else {

// code block executed if condition is false

}

2. switch-case Statement:

  • Provides a convenient way to execute different blocks of code based on the value of an expression.

switch (expression) {

case value1:

// code block executed if expression equals value1

break;

case value2:

// code block executed if expression equals value2

break;

default:

// code block executed if expression doesn't match any case

}

3. while Loop:

  • Executes a block of code repeatedly as long as a specified condition is true.

while (condition) {

// code block executed while condition is true

}

4. do-while Loop:

  • Similar to the while loop, but the code block is executed at least once before the condition is tested.

do {

// code block executed at least once

} while (condition);

5. for Loop:

  • Executes a block of code repeatedly for a specified number of times.

for (initialization; condition; increment/decrement) {

// code block executed while condition is true

}

6. break Statement:

  • Terminates the nearest enclosing loop or switch-case statement.

7. continue Statement:

  • Skips the rest of the current iteration of a loop and continues with the next iteration.

Functions

Functions in the C programming language are reusable blocks of code that perform a specific task. They allow you to break down your program into smaller, manageable parts, making your code more organized, readable, and easier to maintain. Here's some information about functions in C:

  • Syntax: The syntax for defining a function in C is as follows:

return_type function_name (parameter_list) {

// Function body

// Statements

return value; // optional

}

  • Return Type: The return type specifies the data type of the value that the function returns after performing its task. It can be 'void' if the function does not return any value.

  • Function Name: It is the name of the function that is used to call it from other parts of the program.

  • Parameter List: It contains the list of parameters (arguments) that the function accepts. Parameters are optional. If a function does not require any input, the parameter list can be left empty.

  • Function Body: It contains the set of statements that define what the function does when called.

  • Return Statement: It is used to return a value from the function to the calling code. The return statement is optional for functions with a 'return' type of 'void'.

  • Function Declaration: Before you can use a function in C, you must declare it. Function declaration specifies the function's name, return type, and parameter types (if any). It's typically done at the beginning of the program or in a header file.

  • Function Call: To execute a function, you call it by its name followed by parentheses. If the function has parameters, you pass values inside the parentheses.

Array

1. Declaration : To declare an array in C, you specify the data type of the elements and the size of the array.

Syntax:

data_type array_name [array_size];

Example:

int numbers[ 5 ]; // Declares an array of 5 integers

2. Initialization : You can initialize the elements of an array at the time of declaration or later in the program.

Example:

int numbers[ 5 ]; = { 1, 2, 3, 4, 5 }; // Initializes array elements during declaration

3. Accessing Elements: Array elements are accessed using zero-based indexing.

Example:

int third_element numbers[ 2 ]; // Accesses the third element of the array (index 2)

4. Arrays and Pointers: In C, arrays are closely related to pointers. The name of an array acts as a pointer to the first element of the array.

Example:

int numbers[ 5 ];

int *ptr = numbers; // ptr points to the first element of the numbers array

5. Multidimensional Arrays: C supports multidimensional arrays, which are arrays of arrays.

Example:

int matrix[ 3 ] [ 3 ]; // Declares a 3x3 matrix

6. Array Size: The size of an array must be a constant expression known at compile time.

Example:

const SIZE = 5;

int numbers [ SIZE ]; // Declares an array with size determined by a constant expression

7. Array Manipulation: In C, arrays are closely related to pointers. The name of an array acts as a pointer to the first element of the array.

Pointers

1. Declaration: To declare a pointer variable in C, you use the asterisk (*) symbol.

Syntax:

data_type * pointer_name;

Example:

int * ptr; // Declares a pointer to an integer

2. Initialization: Pointers can be initialized with the address of another variable.

Example:

int x = 10;

int * ptr = &x; // Initializes ptr with the address of x

3. Accessing Value: You can access the value stored at a memory address pointed to by a pointer using the dereference operator (*)

Example:

int x = 10;

int * ptr = &x; // Initializes ptr with the address of x

int value = * ptr; // Accesses the value stored at the memory address pointed to by ptr

4. Arrays and Pointers: In C, arrays are closely related to pointers. The name of an array acts as a pointer to the first element of the array.

Example:

int numbers[ 5 ];

int *ptr = numbers; // ptr points to the first element of the numbers array

5. Dynamic Memory Allocation: Pointers are commonly used for dynamic memory allocation using functions like 'malloc()', 'calloc()', and 'realloc()'. These functions allocate memory at runtime and return a pointer to the allocated memory block.

int *ptr = (int * ) malloc( 5 * sizeof ( int) ); // Allocates memory for an array of 5 integers
