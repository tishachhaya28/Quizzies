CSS

What is CSS?

CSS stands for Cascading Style Sheets. It's a language used to style the appearance of HTML elements on web pages.

Why is CSS important?

  • CSS allows web designers to control the layout, colors, fonts, and other visual aspects of a webpage.

  • It separates the content of a webpage from its presentation, making websites easier to maintain and update.

How does CSS work?

  • CSS works by targeting HTML elements and applying styling rules to them.

  • These rules can be defined inline, in a separate CSS file, or internally within the HTML document.

Basic CSS syntax:

  • Selectors: Identify which HTML elements the styles should be applied to.

  • Properties: Define the aspects of the elements you want to style, such as color, size, and position.

  • Values: Specify the specific settings for the properties, like 'red' for color or '20px' for size.

CSS Selectors:

  • Selectors are patterns that match elements in an HTML document. They can be based on element names, classes, IDs, attributes, or relationships between elements.

Example:

Element Selector: `p { color: blue;}`

Class Selector: `.highlight { background-color: yellow; }`

ID Selector: `#header { font-size: 24px; }`

`border:1px solid black;`

Attribute Selector: `input[type="text"] { border: 1px solid black; }`

CSS Properties:

  • Properties are the styling attributes you apply to the selected elements. Each property has a specific value.

Example:

`color`: Specifies the size of the font.

`font-size`: Specifies the background color.

`background-color`: Specifies the border around an element.

`border`:

`margin`,`padding`: Specifies the spacing around an element.

CSS Values:

  • Values are assigned to properties to define how they should be applied. Values can be keywords, numeric values, colors, URLs, etc.

Example:

`color: blue;`

`font-size: 16px;`

`background-color`: #ffffff;`

`border:1px solid black;`

`margin`,`padding`: 10px;`

CSS Selectors

  • Simple selectors (select elements based on name, id, class)
  • Combinator selectors (select elements based on a specific relationship between them)
  • Pseudo-class selectors (select elements based on a certain state)
  • Pseudo-elements selectors (select and style a part of an element)
  • Attribute selectors (select elements based on an attribute or attribute value)

Attribute Selectors:

  • These selectors target elements based on their attributes and attribute values.

  • Examples include [attribute], [attribute=value], [attribute~=value], [attribute|=value], and [attribute^=value].

    1. CSS [attribute="value"] Selector

    2. CSS [attribute~="value"] Selector

    3. CSS [attribute|="value"] Selector

    4. CSS [attribute^="value"] Selector

    5. CSS [attribute$="value"] Selector

    6. CSS [attribute*="value"] Selector

Pseudo-classes:

  • Pseudo-classes select elements based on their state or position.

  • Common pseudo-classes include `:hover`, `:active`, `:focus`, `:first-child`, `:last-child`, `:nth-child()`, `:nth-of-type()`, `:not()`, `:checked`, `:disabled`, and `:enabled.`

    1. Anchor Pseudo-classes

    2. The :first-child Pseudo-class

    3. The :lang Pseudo-class

    4. general sibling selector (~)

Pseudo-elements:

  • Pseudo-elements target certain parts of elements rather than whole elements.

  • Examples include `::before`, `::after`, `::first-line`, and `::first-letter`.

    1. The ::first-letter Pseudo-element

    2. The ::first-line Pseudo-element

    3. The ::before Pseudo-element

    4. The ::after Pseudo-element

    5. The ::marker Pseudo-element

    6. The ::selection Pseudo-element

Combinators:

  • Combinators specify relationships between selectors.

  • Common combinators include descendant combinator (` `), child combinator (`>`), adjacent sibling combinator (`+`), and general sibling combinator (`~`).

    1. descendant selector (space)

    2. child selector (>)

    3. adjacent sibling selector (+)

    4. general sibling selector (~)

CSS Border Style

  • dotted- Defines a dotted border
  • dashed- Defines a dashed border
  • solid- Defines a solid border
  • double- Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge- Defines a 3D ridged border. The effect depends on the border-color value
  • inset- Defines a 3D inset border. The effect depends on the border-color value
  • outset- Defines a 3D outset border. The effect depends on the border-color value
  • none- Defines no border
  • hidden- Defines a hidden border

CSS Overflow

  • visible- Default. The overflow is not clipped. The content renders outside the element's box
  • hidden- The overflow is clipped, and the rest of the content will be invisible
  • scroll- The overflow is clipped, and a scrollbar is added to see the rest of the content
  • auto- Similar to scroll, but it adds scrollbars only when necessary

overflow: visible

  • By default, the overflow is visible, meaning that it is not clipped and it renders outside the element's box:

Example:

// div {

// overflow: visible;

// }

overflow: hidden

  • With the hidden value, the overflow is clipped, and the rest of the content is hidden

Example:

// div {

// overflow: hidden;

// }

overflow: scroll

  • Setting the value to scroll, the overflow is clipped and a scrollbar is added to scroll inside the box. Note that this will add a scrollbar both horizontally and vertically

Example:

// div {

// overflow: scroll;

// }

overflow: auto

  • The auto value is similar to scroll, but it adds scrollbars only when necessary

Example:

// div {

// overflow: auto;

// }

z-index

  • When elements are positioned, they can overlap other elements.

Example:

// img {

// position: absolute;

// left: 0px;

// top: 0px;

// z-index: -1;

// }

float and clear

    The float Property

  • The CSS float property specifies how an element should float.

  • The CSS clear property specifies what elements can float beside the cleared element and on which side.

  • left- The element floats to the left of its container
  • right- The element floats to the right of its container
  • none- The element does not float (will be displayed just where it occurs in the text). This is default
  • inherit - The element inherits the float value of its parent

CSS Masking

  • With CSS masking you create a mask layer to place over an element to partially or fully hide portions of the element.

Example:

// .mask1 {

// -webkit-mask-image: url( );

// mask-image: url();

// -webkit-mask-repeat: no-repeat;

// mask-repeat: no-repeat;

// }
