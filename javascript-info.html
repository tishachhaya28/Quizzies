JavaScript

Introduction

  • JavaScript is a versatile scripting language primarily used for web development. It's known for its ability to add interactivity and dynamic behavior to websites.

  • JavaScript code runs on the client-side, meaning it's executed in the user's web browser, allowing for real-time manipulation of webpage elements.

  • It was initially created to handle client-side validation and interact with HTML elements, but its capabilities have expanded to include server-side development (Node.js) and building desktop and mobile applications.

Differance between Scripting and Programming language

Feature Scripting Language Programming Language
Execution Environment Interpreted, executed directly by an interpreter or runtime environment Can be compiled into machine code or interpreted

Differance between Scripting and Programming language

Scripting language

  • Interpreted, executed directly by an interpreter or runtime environment

  • Automation, web development, rapid prototyping

  • Simpler syntax, higher-level abstractions, multiple paradigms supported

  • Generally slower due to interpretation at runtime, optimizations may improve performance

  • Shorter development cycles, immediate results without compilation

Programming language

  • Can be compiled into machine code or interpreted

  • System software, application development, high-performance computing

  • May have more complex syntax, tailored for specific paradigms

  • Typically faster due to compilation, can utilize various optimizations

  • Longer development cycles, may require compilation and build processes

Variables

  • var : Declares a variable globally or locally to an entire function, regardless of block scope.

  • let : Declares a block-scoped variable. Variables declared with let are limited to the block in which they are defined.

  • const : Declares a constant variable whose value cannot be reassigned. It also has block scope like `let`.

Data Types

Primitive Data Types

  • Number : Represents numeric data, including integers and floating-point numbers.

  • String : Represents textual data enclosed within single `(')` or double `(")` quotes.

  • Boolean : Represents a logical value of true or false.

  • Undefined : Represents an undefined value.

  • Null : Represents an intentional absence of any object value.

  • Symbol (ES6) : Represents a unique identifier.

Complex Data Types

  • Object : Represents a collection of key-value pairs. Objects can contain other objects, arrays, functions, and more.

  • Array :Represents a collection of elements indexed by contiguous integers. Arrays can contain any data type, including other arrays.

Operators

  • JavaScript supports a wide range of operators for performing operations on variables and values. These include arithmetic operators (+, -, *, /), assignment operators (=, +=, -=),

  • Understanding operators is essential for performing calculations, making decisions in code (using conditional statements), and controlling the flow of execution (using loops).

Functions

  • Functions are blocks of reusable code designed to perform a specific task or calculate a value.

  • JavaScript functions can be declared using the `functions` keyword or as arrow functions (introduced in ECMAScript 6).

  • They can accept parameters (inputs) and return values (outputs), making them flexible and versatile building blocks of JavaScript programs.

Example Code

function addNumbers (num1, num2)

{

  // Inside the function, add num1 and num2 together

  let sum num1 + num2 ;

  // Return the result of the addition

  return sum ;

}

Built-in Functions

  • String Functions : These functions operate on strings and can perform tasks such as concatenation, substring extraction, searching for substrings, and converting case.

  • Number Functions : These functions operate on numbers and can perform tasks such as rounding, converting between different number formats, and generating random numbers.

  • Array Functions : These functions operate on arrays and can perform tasks such as adding or removing elements, searching for elements, sorting, and iterating through arrays.

  • Math Functions : These functions are part of the `Math` object and can perform mathematical calculations such as trigonometric functions, logarithms, exponentiation, and more.

Event Handling

  • Event handling is a fundamental aspect of web development that allows JavaScript code to respond to user interactions with a webpage, such as mouse clicks, keyboard inputs, or page loads.

  • JavaScript event handlers are functions that are executed when a specific event occurs on a webpage element, such as a button click or form submission.

  • Event handling enables developers to create interactive and dynamic user experiences by adding behavior to HTML elements and responding to user actions in real-time.

What are Events

  • Events are actions or occurrences that happen in the browser.

  • Examples include clicking a button, pressing a key, or resizing the window.

How Event Handling Works

  • Event handling is about responding to these actions.

  • You can write functions (event handlers) to react when events occur.

Accessing Elements

You can use various methods to access elements in the DOM, such as `getElementById`, `getElementsByClassName` , `getElementsByTagName`, and `querySelector` .

Example

// Access an element by its ID

var elementById = document. getElementById ('myElementId');

// Access elements by class name

var elementsByClass = document. getElementsByClassName ('myClassName');

// Access elements by tag name

var elementsByTag = document. getElementsByTagName ('div');

// Access the first element matching a CSS selector

var elementBySelector = document. querySelector ('.myClass');

Modifying Elements

You can modify various attributes and properties of DOM elements, such as `innerHTML`, `textContent`, `style`, `setAttribute`, and`classList`.

Example

// Change inner HTML content

elementById.innerHTML = '<p>New content'</p>

// Change text content

elementById.textContent = 'New text' ;

// Change CSS styles

elementById.style.color = 'red';

// Add a CSS class

elementById.classList.add elementById('newClass');

// Set an attribute

elementById.setAttribute('data-id', '123');

Creating and Appending Elements

You can create new elements and append them to the DOM using methods like `createElement` and `appendChild`.

Example

// Create a new paragraph element

var newParagraph = document.createElement('p');

// Set its text content

newParagraph.textContent = 'New paragraph' ;

// Append it to an existing element

document.body.appendChild (newParagraph);

Removing Elements
