Beginner Level

HTML

HTML (Hypertext Markup Language) is the standard markup language used to create and design web pages on the internet.

CSS

Style sheet language determining the presentation of HTML documents, crucial for web design.

Scratch

Visual programming language for creating interactive stories and animations, ideal for beginners.

Blockly

Visual programming language simplifying coding with drag-and-drop blocks, suitable for education.

Intermediate Level

JAVA

Cross-platform programming language renowned for its versatility and scalability.

C#

Versatile programming language by Microsoft, known for simplicity and scalability.

JavaScript

Essential scripting language for web development, enabling dynamic website behavior.

Ruby

Dynamic programming language prized for simplicity and productivity, favored for web development.

PHP

PHP web framework known for elegant syntax and powerful features, facilitating efficient development.

Swift

Apple's programming language for iOS, macOS, and beyond, prioritizing safety and performance.

Frameworks

Django(Python)

High-level Python web framework emphasizing rapid development and clean design.

Ruby on Rails (Ruby)

Web development framework focusing on convention over configuration, powered by Ruby.

Express.js (JavaScript - Node.js)

Minimalist Node.js framework for building web applications with JavaScript.

Laravel (PHP)

PHP web framework known for elegant syntax and powerful features, facilitating efficient development.

ASP.NET Core (C#)

Cross-platform framework for building modern web applications with C#, known for scalability and performance.

Angular JS

Comprehensive framework for dynamic web applications, featuring modular architecture and extensive tooling.

React.js (JavaScript)

JavaScript library for building user interfaces, distinguished by its component-based architecture.

Flutter (Dart)

Google's UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.

React Native (JavaScript)

Framework for building native mobile applications using JavaScript and React, ensuring cross-platform development.

NativeScript (JavaScript)

Open-source framework for building native mobile applications using JavaScript, TypeScript, or Angular, promoting cross-platform efficiency.

Unity (C#)

Cross-platform game development engine known for its versatility and intuitive interface.
